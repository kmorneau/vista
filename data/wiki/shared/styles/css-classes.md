# CSS Classes

Vista provides CSS classes for styling components.

## Layout Classes

- **.row**: Display as row

- **.col**: Display as column

- **.center**: Center content

- **.flex**: Flexbox container

## Spacing Classes

- **.m-1** to **.m-5**: Margin

- **.p-1** to **.p-5**: Padding

## Text Classes

- **.text-left**: Left align

- **.text-center**: Center align

- **.text-right**: Right align

- **.bold**: Bold text

- **.italic**: Italic text

## Color Classes

- **.bg-primary**: Primary background

- **.text-primary**: Primary text

- **.text-muted**: Muted text
