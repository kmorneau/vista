# Vista Core Library

The Vista Core Library provides the foundational UI framework for building desktop and web applications using Arturo's VID (Visual Interface Dialect).

## Core Features

- **Declarative UI**: Build user interfaces using the VID syntax

- **Component System**: Rich set of pre-built UI components

- **Layout Engine**: Flexible layout system with rows, columns, and grids

- **Data Binding**: Automatic state synchronization with UI

- **Event System**: Comprehensive event handling model
