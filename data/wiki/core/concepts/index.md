# Core Concepts

Understanding the fundamental concepts of Vista is essential for building robust applications. This section covers the architecture and design principles that underpin the framework.

## Key Concepts

- [VID Overview](/wiki/core/concepts/vid-overview/) - Introduction to the Visual Interface Dialect

- [Face Model](/wiki/core/concepts/face-model/) - Understanding faces and facets

- [Layout System](/wiki/core/concepts/layout-overview/) - Layout concepts and philosophy

- [State Management](/wiki/core/concepts/state-management/) - Managing application state

- [Event System](/wiki/core/concepts/event-system/) - Event handling architecture

## Face Model
The face model is the foundation of Vista's UI system. Faces are the basic building blocks that represent UI elements with properties, state, and behavior.

## Event System
Vista provides a comprehensive event system that allows you to respond to user interactions, from simple clicks to complex gestures.
