# UI Components

Vista provides a comprehensive library of pre-built UI components for building modern user interfaces.

## Component Categories

- [Text Components](/wiki/core/components/text-components/) - Labels, titles, subtitles

- [Button Controls](/wiki/core/components/button-controls/) - Buttons and key buttons

- [Input Fields](/wiki/core/components/input-fields/) - Text inputs, textareas, selects

- [Selection Controls](/wiki/core/components/selection-controls/) - Checkboxes, toggles, radio groups

- [Sliders & Choice](/wiki/core/components/sliders-choice/) - Sliders, rotary, drop lists

- [Containers](/wiki/core/components/containers/) - Boxes, panels, groups

- [Layout Containers](/wiki/core/components/layout-containers/) - Rows, columns, grids

- [Navigation](/wiki/core/components/navigation-components/) - Tabs, toolbar, menubar

- [Data Display](/wiki/core/components/data-display/) - Lists, tables

- [Visual Dividers](/wiki/core/components/visual-divider/) - Dividers, spacers, splits

- [Canvas](/wiki/core/components/canvas-artifacts/) - Canvas elements
