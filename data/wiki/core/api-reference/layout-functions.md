# Layout Functions

Complete reference for Vista's layout control functions.

## Layout Containers

- **across**: Arrange children horizontally

- **below**: Arrange children vertically

- **row**: Row layout container

- **col**: Column layout container

- **grid**: Grid layout container

## Layout Control

- **spacing**: Set space between children

- **padding**: Set internal padding

- **margin**: Set external margin

- **align**: Set horizontal alignment

- **valign**: Set vertical alignment

## Positioning

- **origin**: Set origin position

- **offset**: Set offset position

- **size**: Set component size

## Nesting

- **scope**: Create isolated layout scope

- **panel**: Create panel container

- **group**: Create group container
