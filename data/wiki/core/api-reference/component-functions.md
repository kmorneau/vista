# Component Functions

Complete reference for Vista's component creation functions.

## Text Components

- **title**: Display large heading text

- **subtitle**: Display smaller heading text

- **text**: Display regular text

- **label**: Display label text

## Button Components

- **button**: Create a clickable button

- **key_button**: Create a keyboard button

## Input Components

- **field**: Single-line text input

- **textarea**: Multi-line text input

- **select**: Dropdown selection

## Selection Components

- **checkbox**: Checkbox control

- **toggle**: Toggle switch

- **radio**: Radio button

## Container Components

- **box**: Basic container box

- **panel**: Panel container

- **group**: Grouped container
