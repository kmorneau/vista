# Event Functions

Complete reference for Vista's event handling functions.

## Event Handlers

- **on-click**: Handle click events

- **on-change**: Handle change events

- **on-input**: Handle input events

- **on-over**: Handle mouse enter events

- **on-out**: Handle mouse leave events

- **on-down**: Handle mouse down events

- **on-up**: Handle mouse up events

- **on-key**: Handle keyboard events

- **on-focus**: Handle focus events

- **on-blur**: Handle blur events

## Global Events

- **key**: Global keyboard shortcut

- **on-key-down**: Global key down

- **on-key-up**: Global key up

## Event Control

- **prevent**: Prevent default event behavior

- **stop**: Stop event propagation
