# Vista Graphics Library

The Vista Graphics Library provides powerful 2D drawing capabilities for creating custom graphics, visualizations, and interactive canvas-based applications.

## Graphics Features

- **Drawing Primitives**: Lines, shapes, polygons, paths

- **Transformations**: Rotate, scale, translate, skew

- **Colors & Gradients**: RGB, RGBA, named colors, linear/radial gradients

- **Text Rendering**: Text with fonts and alignment

- **Image Support**: Image drawing, scaling, sprite sheets

- **Animation**: Frame-based animation and transitions
