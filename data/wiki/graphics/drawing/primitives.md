# Drawing Primitives

Reference for Vista Graphics basic drawing primitives.

## Line
`draw canvas [
line 0 0 100 100
]`

## Box (Rectangle)
`draw canvas [
box 10 10 100 100
]`

## Circle
`draw canvas [
circle 100 100 50
]`

## Ellipse
`draw canvas [
ellipse 100 100 50 30
]`

## Arc
`draw canvas [
arc 100 100 50 0 90
]`

## Point
`draw canvas [
point 100 100
]`
