# Typography Tips

Best practices for working with text in Vista Graphics.

## Font Selection

- Use readable fonts for body text (Arial, Helvetica, Georgia)

- Limit font families to 2-3 per design

- Consider web-safe fonts or provide fallbacks

## Font Sizing

- Use relative sizes for accessibility

- Maintain clear hierarchy (title > subtitle > body)

- Avoid very small text (below 10px for body)

## Color Contrast

- Ensure adequate contrast between text and background

- WCAG recommends 4.5:1 ratio for normal text

- Test text readability on different backgrounds

## Spacing

- Use appropriate line height (1.4-1.6 for body text)

- Add space between paragraphs

- Consider letter spacing for headings

## Performance

- Limit custom web fonts

- Preload critical fonts

- Use font-display: swap for fallback
