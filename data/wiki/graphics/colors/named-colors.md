# Named Colors

Vista Graphics supports CSS named colors for easy color specification.

## Basic Colors
`draw canvas [
pen red
fill blue
box 0 0 100 100
]`

## Common Named Colors

- **Primary**: red, blue, green, yellow, orange, purple

- **Neutral**: black, white, gray, grey

- **Status**: success, warning, error, info

## Extended Colors
`draw canvas [
fill tomato
circle 100 100 50
fill steelblue
circle 200 100 50
]`
