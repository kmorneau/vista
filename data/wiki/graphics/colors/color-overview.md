# Color Overview

Vista Graphics provides multiple ways to specify colors for drawing operations.

## Color Specification Methods

- **Named Colors**: CSS color names (red, blue, green)

- **RGB**: Red, Green, Blue values (0-255)

- **RGBA**: RGB with alpha transparency

- **Hex**: Hexadecimal color values (#FF0000)

- **HSL**: Hue, Saturation, Lightness

## Quick Reference
`draw canvas [
pen red
fill blue
box 0 0 100 100
]`

## Color Applications

- **pen**: Outline color

- **fill**: Fill color

- **font-color**: Text color
