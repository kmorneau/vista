# Particle Systems

Vista Graphics supports particle systems for creating visual effects like fire, smoke, and sparks.

## Basic Particle
`particles: []
add-particle particles x y vx vy life color

animate [
update-particles particles
draw-particles particles
]`

## Particle Properties

- **Position**: x, y coordinates

- **Velocity**: vx, vy speed

- **Life**: How long particle lasts

- **Color**: Particle color

- **Size**: Particle size

## Particle Effects

- Fire effect

- Smoke effect

- Sparks effect

- Rain effect

- Snow effect
