# Graphics Functions

Complete reference for Vista Graphics helper functions.

## Drawing Functions

- **draw**: Execute drawing commands

- **clear**: Clear canvas

- **refresh**: Refresh canvas

## Animation Functions

- **animate**: Start animation loop

- **on-frame**: Frame callback

- **stop-animation**: Stop animation

## Utility Functions

- **distance**: Calculate distance between points

- **angle**: Calculate angle between points

- **lerp**: Linear interpolation

- **clamp**: Clamp value to range

## Canvas Functions

- **canvas-size**: Get canvas dimensions

- **resize-canvas**: Resize canvas
