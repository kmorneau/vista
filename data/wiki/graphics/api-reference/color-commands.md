# Color Commands

Complete reference for Vista Graphics color and gradient commands.

## Color Functions

- **rgb**: Create RGB color

- **rgba**: Create RGBA color with alpha

- **hsl**: Create HSL color

- **hsla**: Create HSLA color

## Gradient Functions

- **linear**: Create linear gradient

- **radial**: Create radial gradient

- **conic**: Create conic gradient

## Color Utilities

- **lerpColor**: Interpolate between colors

- **color-to-hex**: Convert to hex

- **hex-to-color**: Convert from hex
