# Draw Commands

Complete reference for Vista Graphics draw commands.

## Shape Commands

- **line**: Draw a line

- **box**: Draw a rectangle

- **circle**: Draw a circle

- **ellipse**: Draw an ellipse

- **polygon**: Draw a polygon

- **arc**: Draw an arc

- **point**: Draw a point

## Path Commands

- **path**: Create a path

- **move**: Move to position

- **line**: Draw line to position

- **curve**: Draw curve

- **close**: Close path

## State Commands

- **pen**: Set pen color/style

- **fill**: Set fill color/style

- **push**: Save state

- **pop**: Restore state
