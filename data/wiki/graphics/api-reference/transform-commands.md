# Transform Commands

Complete reference for Vista Graphics transform commands.

## Transform Functions

- **translate**: Translate coordinate system

- **rotate**: Rotate coordinate system

- **scale**: Scale coordinate system

- **skew**: Skew coordinate system

## Matrix Operations

- **reset-transform**: Reset transformation matrix

- **save-transform**: Save transformation matrix

- **restore-transform**: Restore transformation matrix

## Coordinate System

- **coord-system**: Set coordinate system type

- **origin**: Set origin point
