# Skew

Skew (shear) subsequent drawing operations.

## Basic Skew
`draw canvas [
skew 20 0
box 0 0 100 100
]`

## Parameters

- **x-skew**: Horizontal skew angle in degrees

- **y-skew**: Vertical skew angle in degrees

## Combined Skew
`draw canvas [
skew 15 10
box 0 0 100 100
]`
